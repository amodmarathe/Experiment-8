 Aim

---

Theory

A **for loop** in Python is used to iterate over a sequence such as a list, tuple, string, or range of numbers. It is mainly used when the **number of iterations is known**.

The `range()` function is commonly used with a for loop to generate a sequence of numbers.

In Python, nested for loops can also be used to work with matrices, patterns, and complex calculations.
